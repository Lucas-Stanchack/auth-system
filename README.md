#  Sistema de Autenticação + Kanban

Sistema full stack com autenticação JWT e gerenciamento de tarefas estilo Kanban.

---

##  Funcionalidades

- Login com JWT
- Senha criptografada com bcrypt
- Rotas protegidas (Auth Middleware)
- Interceptor Angular
- Controle de acesso por usuário

---

##  Kanban (em desenvolvimento)

- Criar tarefas
- Listar tarefas por usuário
- Atualizar status (pendente, andamento, concluída)
- Excluir tarefas

---

##  Tecnologias

### Frontend
- Angular
- TypeScript

### Backend
- Node.js
- Express
- MySQL
- JWT
- bcrypt
